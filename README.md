                                       Web Teknolojileri Projesi 



##  Öne Çıkan Özellikler

* **Dark Mode:**
JavaScript ile geliştirilmiş ve `localStorage` kullanılarak tüm sayfalarda hafızada tutulan tema sistemi.

* **Canlı API Entegrasyonu:**
OMDb API kullanılarak JavaScript `fetch` metodu ile geliştirilmiş çalışan bir "Film Rehberi" arama motoru.

* **Kullanıcı Giriş Sistemi:**
E-posta ve şifre doğrulaması yapan, form kontrollü basit simüle edilmiş giriş/çıkış paneli.

* **Responsive Tasarım:**
Tüm cihazlarda (Mobil/Tablet/PC) sorunsuz çalışan Bootstrap 5 ızgara (grid) sistemi.

* **Dinamik Navigasyon:**
Tüm sayfalar arasında kesintisiz geçiş sağlayan, aktif giriş/çıkış butonlarına sahip menü mimarisi.

##  Sayfa Yapısı

1. **Hakkımda (`index.html`):** Kişisel tanıtım ve ana sayfa.
2. **Özgeçmiş (`cv.html`):** Eğitim bilgileri ve yeteneklerin listelendiği tablo yapısı.
3. **Şehrim (`sehir.html`):** Memleket tanıtımı ve görsel galeri.
4. **Mirasımız (`miras.html`):** Şehrin kültürel ve tarihi mirasının detaylı anlatımı.
5. **Film Rehberi (`film.html`):** API üzerinden asenkron veri çeken arama sayfası.
6. **İletişim (`iletisim.html`):** JavaScript ile boş geçilemez kuralları (form validation) eklenmiş iletişim formu.
7. **Giriş Yap (`login.html`):** Öğrenci giriş paneli.

##  Kullanılan Teknolojiler

* **HTML5** (Semantik etiket yapısı)
* **CSS3 & Bootstrap 5** (Modern ve mobil uyumlu tasarım)
* **JavaScript (Vanilla JS)** (Dark mod, API işlemleri ve form kontrolleri)
